# Movie Streaming App Documentation

## Overview
A React + TypeScript movie streaming app using TMDB API. Features authentication, movie browsing, search, details, and favorites.

## Features
- Social and email/password login (localStorage)
- Browse: popular, now playing, upcoming, top-rated
- Search movies
- Movie details (cast, trailer, etc.)
- Favorites management

## Architecture
- `src/components/`: UI components
- `src/api/`: API utilities
- `src/hooks/`: Custom hooks

## How to Run
1. Install dependencies: `npm install`
2. Add your TMDB API key in `src/api/tmdb.ts`
3. Start: `npm run dev`

## Deployment
- Deploy on Netlify, Vercel, or Firebase

## Customization
- Add dark/light mode, i18n, or Firebase integration as needed.
